# VIM Setting


## Global Setting .vimrc

```bash
copy colors/solarized.vim to ~/.vim/colors/solarized.vim  
copy vimrc to ~/.vimrc
```

## CERN Root C-type Setting

```bash
copy syntax/c.vim to ~/.vim/syntax/c.vim
```

## Fast Setting

```bash
ln -s AMSProject/sw/vim ~/.vim
ln -s AMSProject/sw/vim/.vimrc ~/.vimrc
```
